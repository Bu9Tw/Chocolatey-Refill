# 使用方式

先確定 Windows Update 都更新完，不然裝 SSMS 會失敗

以工作管理員開啟 Cmd

輸入 PowerShell.exe -ExecutionPolicy UnRestricted -File {Refill.ps1 的實體路徑}

## 選擇安裝

### Vue.js

```
npm install -g vue
npm install -g @vue/cli
npm install -g @vue/cli-service-global
```

### Angular

```
npm install -g @angular/cli
```

### VSCode

